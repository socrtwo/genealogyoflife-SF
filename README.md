# Genealogy of Life

Converts the Catalogue of Life species database into GEDCOM genealogy format.

Includes the 2008 Catalogue of Life dataset converted to GEDCOM format.

**Note:** The database file (108 MB) is too large for a git repository. Download it from the [Releases](https://github.com/socrtwo/genealogyoflife-SF/releases) page.
